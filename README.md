# Career Copilot Agent

AI-powered application that analyzes a candidate's CV against a Job Description and generates a fit score, missing skills, CV suggestions, and a learning roadmap.

- [MVP Scope](docs/MVP_SCOPE.md)
- [Architecture Draft](docs/ARCHITECTURE_DRAFT.md)
- [API Draft](docs/API_DRAFT.md)
- [Changelog](CHANGELOG.md)

## Tech Stack

- Frontend: Next.js
- Backend: FastAPI
- Agent: LangGraph
- Database: Supabase
- Deployment: Vercel + AWS
- AI API: Gemini
